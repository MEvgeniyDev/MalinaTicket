# Журнал изменений

## 26.5.9 - 2026-05-26

- Добавлена универсальная поддержка Paper/Purpur `1.16.5-26.1.2`.
- Целевой байткод плагина снижен до Java 16, при этом плагин продолжает работать на более новых версиях Java.
- Базовый API для сборки заменен на Paper API `1.16.5-R0.1-SNAPSHOT`.
- Добавлены адаптеры совместимости для MiniMessage/чата, GUI-текста, свечения предметов, `custom-model-data` и звуков.
- Adventure platform добавлен внутрь JAR и перенесен в пакет плагина, чтобы кликабельные сообщения работали на старых серверах.
- Формат YAML-хранилища тикетов и папка `tickets/backups` оставлены совместимыми с предыдущей версией.

## 26.5.8 - 2026-05-24

- GUI разделен на отдельные классы меню.
- Улучшена настройка GUI-предметов.
- Добавлены `amount`, `glow`, `custom-model-data`.
- Улучшено безопасное сохранение YAML.
- Добавлены тесты.
- Обновлены манифест и контрольная сумма.

## 26.5.7 - 2026-05-22

- Первый упакованный релиз для Paper/Purpur `26.1.2`.
- Добавлено создание тикетов игроками, меню персонала, назначение ответственного, закрытие, переоткрытие, мягкое удаление, окончательное удаление, телепорт, офлайн-уведомления и YAML-хранилище.
- Добавлены настраиваемые GUI, сообщения, категории, справочник прав, задержки создания тикетов, лимиты открытых тикетов и русские UTF-8 конфиги по умолчанию.
- Добавлены данные релиза в манифест JAR-файла и SHA-256 контрольная сумма при сборке.

## Примечания

- Версия `26.5.9` является универсальной сборкой для Paper/Purpur `1.16.5-26.1.2`.
- Версия `26.5.8` остается предыдущей сборкой только для современной ветки Paper/Purpur `26.1.2`.
- Данные тикетов хранятся в `plugins/MalinaTicket/tickets`; перед обновлениями лучше делать копию этой папки.
- Миграция базы данных не нужна: текущий публичный формат хранения остается YAML.

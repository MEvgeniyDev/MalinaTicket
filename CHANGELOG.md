# Changelog

## 26.5.8 - 2026-05-24

- GUI разделен на отдельные классы меню.
- Улучшена настройка GUI-предметов.
- Добавлены `amount`, `glow`, `custom-model-data`.
- Улучшено безопасное сохранение YAML.
- Добавлены тесты.
- Обновлены manifest/checksum.

## 26.5.7 - 2026-05-22

- First packaged release for Paper/Purpur 26.1.2.
- Added player ticket creation, staff review menus, assignment, closing, reopen, soft delete, purge, teleport, offline notifications, and YAML storage.
- Added configurable GUI, messages, categories, permissions reference, cooldowns, open-ticket limits, and UTF-8 Russian defaults.
- Added release metadata in the JAR manifest and a SHA-256 checksum generated by the build.

## Notes

- Requires Java 25 and Paper/Purpur 26.1.2.
- Ticket data is stored in `plugins/MalinaTicket/tickets`; back this directory up before updates.
- No database migration is needed for 26.5.7 because this is the first public storage format.
